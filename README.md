# 🐼 Pandas & NumPy — Data Analysis Practice

A collection of Jupyter notebooks covering core **Pandas** and **NumPy** concepts for data analysis, built with real-world datasets.

---

## 📁 Repository Structure

```
├── 📓 Notebooks
│   ├── pandas.ipynb                              # Pandas fundamentals & introduction
│   ├── Operations_Pandas.ipynb                   # Data operations in Pandas
│   ├── FeatureExtraction_Pandas.ipynb            # Feature extraction techniques
│   ├── GroupByAggregation_Pandas.ipynb           # GroupBy and aggregation methods
│   ├── MissingData_Pandas.ipynb                  # Handling missing/null data
│   ├── Merging_Joining_Concatination_Pandas.ipynb # Merging, joining & concatenating DataFrames
│   ├── PivotTables_Pandas.ipynb                  # Pivot tables and cross-tabulations
│   ├── Countries.ipynb                           # Analysis on Countries dataset
│   ├── numpyArray.ipynb                          # NumPy arrays — core concepts
│   └── numpyArrayExercise.ipynb                  # NumPy array exercises & practice
│
└── 📊 Datasets
    ├── Countries.csv                             # Country-level data
    └── anime.csv                                 # Anime dataset
```

---

## 📚 Topics Covered

### 🐼 Pandas
| Notebook | Topics |
|---|---|
| `pandas.ipynb` | Series, DataFrames, indexing, slicing |
| `Operations_Pandas.ipynb` | Arithmetic, filtering, sorting, applying functions |
| `FeatureExtraction_Pandas.ipynb` | Creating new features, transformations |
| `GroupByAggregation_Pandas.ipynb` | `groupby()`, `agg()`, `transform()`, `apply()` |
| `MissingData_Pandas.ipynb` | `isnull()`, `dropna()`, `fillna()`, imputation |
| `Merging_Joining_Concatination_Pandas.ipynb` | `merge()`, `join()`, `concat()`, different join types |
| `PivotTables_Pandas.ipynb` | `pivot_table()`, `crosstab()`, reshaping data |
| `Countries.ipynb` | End-to-end analysis on the Countries dataset |

### 🔢 NumPy
| Notebook | Topics |
|---|---|
| `numpyArray.ipynb` | Array creation, indexing, slicing, broadcasting |
| `numpyArrayExercise.ipynb` | Hands-on exercises and problem solving |

## 🚀 Getting Started

### Prerequisites

Make sure you have Python 3.x installed along with the following libraries:

```bash
pip install pandas numpy jupyter matplotlib
```

### Run Locally

```bash
# Clone the repository
git clone https://github.com/sakshidangi2006/<repo-name>.git

# Navigate into the project directory
cd <repo-name>

# Launch Jupyter Notebook
jupyter notebook
```

Then open any `.ipynb` file from the Jupyter interface in your browser.
